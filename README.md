# Health CheckUp

Health CheckUp is a simple web-based recovery guide application that allows users to:

- Create an account and log in
- Manage their profile
- Generate a personalized post-hospital recovery plan
- View diet recommendations
- See warning signs and follow-up reminders

The project uses a clean responsive UI with authentication support connected to a backend API.

---

## Features

### Authentication
- User registration
- User login
- JWT token-based authentication
- Persistent login using localStorage

### Profile Management
- Edit user profile
- Update name, email, and mobile number

### Recovery Plan Generator
Users can select:
- Surgery / condition type
- Age group
- Discharge date
- Existing medical conditions

The app then generates:
- Day-by-day recovery schedule
- Recommended foods
- Things to avoid
- Warning signs
- Follow-up reminders

### Responsive Design
- Mobile friendly
- Tablet friendly
- Desktop optimized

### Notifications
- Toast notifications using Toastify.js
